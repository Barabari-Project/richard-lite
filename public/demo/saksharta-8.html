Understanding Basics of Web Dev

Over the next hour we will be making this fanpage for Virat Kohli using some basic programming concepts. We'll primarily use HTML and CSS

<html>: This is like the main container for the webpage. It holds all the content that we want to show.


<body>: This is where we put all the things that we want to show on the webpage. The computer reads this part and displays it on the screen.


<h1>: This is the biggest heading on the webpage. It tells us the main title, which is "Virat Kohli"

<h2>: These are like smaller headings that tell us different sections of the webpage. For example, "About Virat Kohli", "Career Achievements", and "Personal Life" are different sections


<p>: These are like paragraphs of text that tell us information about Virat Kohli. It's like a story that explains who he is and what he has achieved

<ul>: This is a list where we can put some important points. Each point is marked with a dot, and we call it a "bullet point". In this case, it lists some of Virat Kohli's career achievements.


<li>: These are the individual bullet points inside the list. Each point represents something special that Virat Kohli has achieved.

Let's bring this all together! We can put all the code we've learned so far into one webpage

Let's now add some styling to this

<style>: This is where we can add some styling to our webpage. We can change the font, background color, and text color etc of our webpage.


h1 {}: This part is about styling a specific type of heading, the h1 heading. It's like giving special instructions just for the big and important headings on the page


background-color:white: This is like picking the colour of the background of the webpage. It says that the background color of the whole webpage should be white


color:white: This says that the text color inside the h1 heading should be white. Since the background is black, using white text makes it easy to read

text-align:center: This is about how the text inside the h1 heading is aligned. It says that the text should be centered, which means it will be in the middle of the heading and look neat
